# Українізатор DSM 🇺🇦

Апка для Synology DSM, яка перекладає інтерфейс системи (і встановлених пакетів)
українською. Ставиш галочки — що саме перекласти — і тиснеш кнопку. Так само легко
відкочується назад.

---

## Як це працює (коротко)
- DSM зберігає тексти інтерфейсу у файлах локалі `rus`. Апка підміняє їхній **вміст**
  українським перекладом (код локалі лишається `rus` — інакше DSM ламає UI).
- Оригінали зберігаються назавжди (`.orig`), тож **відкат — байт-в-байт**.
- Привілейовану роботу (запис у системні файли) робить окремий root-демон; веб-морда
  спілкується з ним через захищений канал nginx (тільки залогінений адмін DSM + пароль).

## Встановлення

### Крок 1. Встанови пакет
1. Відкрий **DSM** у браузері (як адмін).
2. **Центр пакетів** → **Ручне встановлення**.
3. Вибери файл `UkrainianL10n.spk` → **Далі** → **Готово**.
4. Погодься на попередження про невідомого видавця — це нормально.

### Крок 2. Один раз увімкни «двигун» (root-крок) ⚠️
Робиться **лише раз**. Без цього кнопки апки не працюють — це безпека Synology
(непідписаному пакету root не дають, і це не обійти).

**Через Планувальник завдань (без командного рядка):**
1. **Панель керування** → **Планувальник завдань**.
2. **Створити** → **Запланована задача** → **Скрипт користувача**.
3. Вкладка «Загальне»: назва `ukr-l10n-engine`, користувач — **root**.
4. Вкладка «Параметри завдання», поле «Команда»:
   ```
   sh /var/packages/UkrainianL10n/target/bin/ua-l10n-bootstrap.sh install
   ```
5. **OK** → вибери задачу → **Виконати**.

**Або через SSH (той самий рядок):**
```
sudo sh /var/packages/UkrainianL10n/target/bin/ua-l10n-bootstrap.sh install
```

### Крок 3. Користуйся
1. Відкрий апку **«Українізатор DSM»**.
2. Постав галочки на тому, що хочеш перекласти (ядро DSM + пакети).
3. **«Українізувати вибране»** → введи пароль адмін-акаунта DSM.
4. Дивись живий лог — що і де патчиться.
5. Натисни **Ctrl+F5** у браузері, щоб DSM перемалював інтерфейс.

**Відкат:** кнопка **«Повернути оригінал»** — повертає рос. файли байт-в-байт.

## Корисно знати
- **Після оновлень DSM** переклад може злетіти — апка сама його повертає (фоновий
  таймер раз на ~15 хв). Нічого робити не треба.
- **Root-крок** робиться один раз на цьому NAS. Поставив апку наново — повтори його.
- Зняти «двигун»: `sudo sh /var/packages/UkrainianL10n/target/bin/ua-l10n-bootstrap.sh remove`.
